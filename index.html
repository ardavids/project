Tips Jadi Web Developer..

Kami akan memberikan tips agar kalian bisa menjadi seorang Web Developer

Skill Yang Dibutuhkan

Html

HTML(Hypertext Markup Language) memungkinkan seorang pengguna dapat membuat dan menyusun bagian heading, paragraf, link atau tautan, dan blockquote untuk halaman sebuah website.

Css

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan layout sebuah halaman web.

Java Script

JavaScript adalah bahasa pemrograman yang digunakan dalam pengembangan website agar lebih dinamis dan interaktif.

Jquery

jQuery adalah sebuah library JavaScript yang dirancang untuk menyederhanakan traversal dan manipulasi DOM HTML, serta penanganan acara, animasi CSS, dan Ajax.

Bootstrap

Bootstrap adalah framework web development berbasis HTML, CSS, dan JavaScript yang dirancang untuk mempercepat proses pengembangan web responsive dan mobile-first

Tailwind

Tailwind adalah sebuah framework CSS yang memberikan sekumpulan peraturan siap pakai, dalam bentuk class yang bisa langsung diaplikasikan di sebuah elemen HTML

git

GIT merupakan kepanjangan dari Group Inclusive Tour, adalah sebuah version control yang banyak digunakan oleh programmer untuk berkolaborasi untuk pembuatan dan pengembangan aplikasi.

Github

GitHub adalah platform pengembangan perangkat lunak berbasis cloud yang menggunakan sistem Git untuk menyimpan, mengelola, melacak, dan mengontrol perubahan pada kode.

Rekomendasi Kursus

Dicoding Indonesia

Dicoding adalah sebuah perusahaan startup yang bertujuan untuke mengembangkan ekosistem Developer di Indonesia. Dicoding juga memiliki platform pembelajaran elektronik. Selain itu dicoding juga sebagai satu-satunya Google Developers Authorized Training Partner di Indonesia dan menjadi mitra Badan Ekonomi Kreatif (BEKRAF) dalam menyelenggarakan program pembangunan ekonomi kreatif digital seperti Bekraf Developer Day. Dicoding adalah salah satu dari 20 anggota kelompok kerja ASEAN yang menciptakan rencana Future ASEAN Agenda for TVET (Technical and Vocational Education Training).
